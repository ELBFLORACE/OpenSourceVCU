## Intention
This package is fully generated by the [`vcu_can_generator`](../vcu_can_generator/) package and contains all the available CAN Messages as ROS messages.
For the most part these messages only get published by the mailman ( under the topic `/sensor/{canName}/{messageName}`, for example `/sensor/can1/pressure_sus` ) and used by the monitoring backend.

The messages in this package are not guaranteed to stay the same forever so use them with caution.
All files were generated at commit `110b681f909421582b4ad1722b704d35762616e3`


## Usage
### Examples
```cpp
#include "vcu_can_msgs_yourcar17/msg/can4_gps_rear_velocities.hpp"

auto msg = vcu_can_msgs_yourcar17::msg::Can4GpsRearVelocities();
```
